# 🤖 AI Resume ATS Assistant (RAG Based)

An AI-powered Resume ATS Assistant built using RAG (Retrieval-Augmented Generation), LangChain, ChromaDB, Streamlit, and Groq API.

This project analyzes resumes intelligently using semantic search and Large Language Models (LLMs) to generate ATS scores, candidate analysis, and job matching insights.

---

# 🚀 Features

✅ Resume Upload (PDF)

✅ RAG-based Semantic Search

✅ ATS Match Score Calculation

✅ AI Candidate Analysis

✅ Job Description Matching

✅ Skill Gap Detection

✅ Vector Database using ChromaDB

✅ LLM-powered Responses using Groq + Llama 3

✅ Streamlit Web Interface
